# Canvas New UI v2 - 2015

The new Canvas User Interface (UI) is an updated interface structure enhanced by a responsive user experience for the Dashboard and Global Navigation menu. The Course Navigation menu also collapses on smaller screens, allowing for a more content-focused and responsive layout.
